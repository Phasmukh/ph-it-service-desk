# PH IT Services – Internal Service Desk Portal

## Overview

A simple IT Service Desk Incident Management System built using Python and Streamlit.

The application allows first-line support analysts to:

- Log incidents
- Generate ticket IDs
- Assign priorities
- View troubleshooting steps
- Save tickets
- Search tickets
- Download ticket data

## Technologies

- Python
- Streamlit
- Pandas

## Features

- Incident logging
- Ticket management
- CSV storage
- Dashboard
- Search
- Download CSV
